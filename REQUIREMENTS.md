# Virtual environment
The virtual environment based on Python 3.12.6

# Python packages
See in requirements.txt
To install the easily, run 'pip install -r requirements.txt' command in the terminal (while you are in the venv created for the project)

# Data files
You will need to create a folder named 'datasets'. You can ask me for the Dnister river network data, or the dummy graph data, and/or you can download hydrorivers datasets from https://www.hydrosheds.org/products/hydrorivers.
Your folder structure should look like this:

Sensor-Placement-Optimization
|-- ...
|-- datasets
|   |-- Dnister river network
|   |-- Dummy graph
|   |-- HydroRIVERS_v10_gr.gdb
|-- ...
